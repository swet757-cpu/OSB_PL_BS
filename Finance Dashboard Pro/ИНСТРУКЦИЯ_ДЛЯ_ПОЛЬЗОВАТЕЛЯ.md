# Пошаговая инструкция для пользователя

Эта инструкция поможет запустить веб-приложение `Finance Dashboard Pro` через Google Apps Script.

## 1. Подготовьте Google Таблицу

Откройте Google Таблицу, где будут храниться финансовые операции.

Создайте лист:

`База_операций`

В первой строке должны быть колонки:

| Колонка | Название |
|---|---|
| A | Дата |
| B | Тип отчета |
| C | Операция |
| D | Статья |
| E | Кошелек |
| F | Направление |
| G | Сумма с НДС |
| H | НДС |
| I | Сумма без НДС |
| J | Комментарий |

Важно: контрагентов добавлять не нужно.

## 2. Заполните данные

В лист `База_операций` добавляйте операции со второй строки.

Для ДДС используйте:

- `Тип отчета`: `ДДС`
- `Операция`: `Поступление` или `Списание`
- сумма берется из колонки `Сумма с НДС`

Для ОПиУ используйте:

- `Тип отчета`: `ОПиУ`
- `Операция`: `Доход` или `Расход`
- сумма берется из колонки `Сумма без НДС`

## 3. Откройте Apps Script

В Google Таблице нажмите:

`Расширения` → `Apps Script`

Откроется редактор Google Apps Script.

## 4. Создайте файлы проекта

В редакторе Apps Script создайте файлы:

- `Code.gs`
- `Index.html`
- `Styles.html`
- `JavaScript.html`

## 5. Скопируйте код

Откройте папку проекта на компьютере:

`c:\Users\Анна\Desktop\тестовая\Finance Dashboard Pro`

Скопируйте содержимое файлов:

- из `Code.gs` в файл `Code.gs`
- из `Index.html` в файл `Index.html`
- из `Styles.html` в файл `Styles.html`
- из `JavaScript.html` в файл `JavaScript.html`

После копирования нажмите `Сохранить`.

## 6. Настройте доступ

В Apps Script нажмите:

`Развернуть` → `Новое развертывание`

Далее:

1. Нажмите на значок шестеренки.
2. Выберите тип: `Веб-приложение`.
3. В поле `Выполнять от имени` выберите `Меня`.
4. В поле `У кого есть доступ` выберите нужный вариант:
   - `Только я`, если дашборд нужен только вам;
   - `Все, у кого есть ссылка`, если хотите дать доступ другим.
5. Нажмите `Развернуть`.

При первом запуске Google попросит разрешения. Их нужно подтвердить, потому что приложение читает данные из вашей Google Таблицы.

## 7. Откройте дашборд

После развертывания Google покажет ссылку на веб-приложение.

Откройте эту ссылку в браузере.

Вы увидите меню:

- `ДДС`
- `ОПиУ`
- `Графики`
- `Заметки`

Разделов `Обзор` и `Контрагенты` в приложении нет.

## 8. Как пользоваться

В верхней панели можно выбрать:

- дату с;
- дату по;
- кошелек;
- направление.

После выбора нажмите:

`Обновить данные`

Дашборд перечитает Google Таблицу и обновит карточки, графики и заметки.

## 9. Как добавлять заметки

Откройте раздел:

`Заметки`

Заполните:

- дату;
- раздел;
- текст заметки.

Нажмите:

`Сохранить заметку`

Заметка появится в Google Таблице на листе `Заметки`.

Если листа `Заметки` еще нет, приложение создаст его автоматически.

## 10. Частые ошибки

Если появляется сообщение:

`Не найден лист База_операций`

Проверьте, что лист называется точно:

`База_операций`

Если появляется сообщение:

`Нет данных за выбранный период`

Проверьте:

- есть ли строки с операциями;
- правильно ли указаны даты;
- не слишком ли узкий выбран период;
- выбран ли правильный кошелек или направление.

Если графики пустые, проверьте значения в колонках:

- `Тип отчета`
- `Операция`
- `Сумма с НДС`
- `Сумма без НДС`

## 11. Важные правила заполнения

Пишите значения одинаково:

- `ДДС`
- `ОПиУ`
- `Поступление`
- `Списание`
- `Доход`
- `Расход`

Если написать с ошибкой, операция может не попасть в нужный расчет.

## 12. Что можно менять

Можно свободно менять:

- статьи;
- кошельки;
- направления;
- комментарии;
- суммы;
- даты.

Их не нужно заранее прописывать в коде. Дашборд берет их из таблицы автоматически.
