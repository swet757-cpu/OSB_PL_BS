# Finance Dashboard Pro

Google Apps Script веб-приложение для универсального финансового дашборда на базе Google Sheets.

## Разделы

- ДДС
- ОПиУ
- Графики
- Заметки

Разделы `Обзор` и `Контрагенты` не используются.

## Структура Google Sheets

Основной лист: `База_операций`

Колонки:

| Колонка | Название |
|---|---|
| A | Дата |
| B | Тип отчета |
| C | Операция |
| D | Статья |
| E | Кошелек |
| F | Направление |
| G | Сумма с НДС |
| H | НДС |
| I | Сумма без НДС |
| J | Комментарий |

Дополнительный лист: `Заметки`

Колонки:

| Колонка | Название |
|---|---|
| A | Дата |
| B | Раздел |
| C | Заметка |
| D | Автор / источник |

Если лист `Заметки` отсутствует, он будет создан при сохранении первой заметки.

## Как установить в Google Apps Script

1. Откройте Google Sheets с листом `База_операций`.
2. Выберите `Расширения` → `Apps Script`.
3. Создайте файлы:
   - `Code.gs`
   - `Index.html`
   - `Styles.html`
   - `JavaScript.html`
4. Скопируйте содержимое файлов из этой папки в одноименные файлы Apps Script.
5. При необходимости скопируйте настройки из `appsscript.json` в файл манифеста.
6. Нажмите `Развернуть` → `Новое развертывание`.
7. Тип развертывания: `Веб-приложение`.
8. Выполнять от имени: `Меня`.
9. Доступ: выберите нужный вариант доступа.
10. Сохраните и откройте ссылку веб-приложения.

## Логика расчетов

ДДС:

- Поступления = `Сумма с НДС`, где `Тип отчета = ДДС` и `Операция = Поступление`
- Списания = `Сумма с НДС`, где `Тип отчета = ДДС` и `Операция = Списание`
- Чистый денежный поток = поступления минус списания

ОПиУ:

- Выручка = `Сумма без НДС`, где `Тип отчета = ОПиУ` и `Операция = Доход`
- Расходы = `Сумма без НДС`, где `Тип отчета = ОПиУ` и `Операция = Расход`
- Прибыль = выручка минус расходы
- Рентабельность = прибыль / выручка * 100

Если выручка равна 0, рентабельность показывается как `0,0%`.

## Универсальность

В коде не зашиты компании, контрагенты, статьи, кошельки или направления.

Статьи, кошельки и направления берутся динамически из листа `База_операций`.
