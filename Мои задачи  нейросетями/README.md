# Календарь финансиста

Одностраничный календарь для задач финансиста: месячный календарь для ориентира, рабочая доска на две недели, быстрое добавление задач, перенос задач между днями и отметка выполнения.

## Как открыть

1. Откройте папку `Мои задачи  нейросетями`.
2. Дважды нажмите на файл `index.html`.
3. Календарь откроется в браузере.

Если страница уже открыта, после изменений нажмите `F5`.

## Что можно делать

- добавлять задачу с датой, временем и типом;
- переносить задачу мышкой на другой день;
- нажимать `Готово`, чтобы отмечать выполнение;
- нажимать `Править`, чтобы изменить название, дату, время, тип или заметку;
- удалять ненужные задачи;
- смотреть результат выполнения рядом с кнопкой `Сегодня`.

## Файлы

- `index.html` — сама страница календаря;
- `styles.css` — внешний вид;
- `script.js` — работа задач, перенос, сохранение и статистика;
- `README.md` — описание проекта.

## Сохранение

Задачи сохраняются в браузере на этом компьютере. Если открыть календарь в другом браузере, список задач может быть пустым.
